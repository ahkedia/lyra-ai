# Eval Dashboard — moved

The live eval dashboard is now generated by `evals/dashboard/build-eval-dashboard.js`
on every eval run (run-evals.sh Step 5c) → `/var/www/lyra-evals/index.html`.

It shows the **change → eval-delta timeline** (each run's pass-rate + per-tier deltas
joined with the commits that landed between runs, annotated with the eval-coverage gate
decision), a pass-rate trend table, the latest failure taxonomy, and the
**days-since-last-successful-run** health metric.

This static markdown stub is retained only so old links resolve. Do not hand-edit eval
metrics here.
