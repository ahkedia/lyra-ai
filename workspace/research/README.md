# Research helper (directory)

This folder holds `research.py` (and related scripts). It is a **directory**, not a single file.

**Do not** use the `read` tool on the folder path `workspace/research` or `/root/.openclaw/workspace/research` — that triggers `EISDIR`.

**Run** research instead:

```text
python3 /root/.openclaw/workspace/research/research.py "your search topic"
```

Alternate copy under the repo: `/root/lyra-ai/research/research.py`.
