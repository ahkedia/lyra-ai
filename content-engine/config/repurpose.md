# Repurposing Chain

## Order

1. **X** — Forces brevity, sharpens core point
2. **LinkedIn** — Same topic, personal narrative layer + professional framing. NOT a reformatted X post — a different angle into the same insight.
3. **Newsletter** (future) — Deepest version, behind-the-scenes context

## The Litmus Test

Stolen from Nikunj/Andrew's actual practice:

> If someone follows you on both X and LinkedIn, would seeing both feel redundant? If yes, you reformatted. If no, you rethought.

## LinkedIn Differences

- More context is OK (1000-2000 chars vs 280)
- First-person narrative tone
- "Here's what I learned" framing works here
- Still no hashtags spam, max 3 if any
- CTA can work here ("have you seen this pattern?")

## Newsletter (when enabled)

- Deep dive: 800-1500 words
- Behind-the-scenes: why this matters to you personally
- Links to sources, related reading
- Subscriber-only examples/data when relevant
