# GitHub Integration

Manage lyra-ai repository and enable self-deployment workflows.

## Configured Repository

- **Repo**: github.com/ahkedia/lyra-ai
- **Branch**: main
- **Token**: Stored in openclaw.json (write:repo scope)

## Capabilities

- Create commits automatically
- Push changes to main
- Trigger GitHub Actions workflows
- Create pull requests
- View commit history

## Usage (via Telegram)

"Lyra, commit changes to SOUL.md" → Stages, commits, pushes to GitHub
"Show me recent commits" → Lists last 10 commits
"Create a PR for X" → Opens pull request with description
